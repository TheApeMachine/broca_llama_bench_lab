\section{Methods}
\label{sec:methods}

\subsection{Architectural Philosophy: The Frozen Language Organ}

The central design decision in \textsc{Mosaic} is the complete separation of
language generation from cognition.  The LLM's weights are \emph{never updated}.
This is not an engineering shortcut to avoid training cost---it is a deliberate
architectural choice that prevents catastrophic forgetting.  When a model's
weights are fine-tuned to incorporate a new fact, existing knowledge degrades
(McCloskey \& Cohen, 1989; Kirkpatrick et al., 2017).  By injecting all new
knowledge through grafts on the residual stream, the base model's capabilities
remain bit-for-bit identical while the substrate accumulates unbounded new
knowledge.

The LLM is thus \textbf{demoted to a speech interface}---an associative cortex
locked in a glass box.  The trainable grafts are the frontal cortex, slipping
intelligent notes through the vents of the residual stream.  When the substrate
discovers a concept the LLM has no vocabulary for (via Hebbian
orthogonalization and Gram--Schmidt), the graft forces the LLM to \emph{invent a
metaphor}---to describe the negative space of the novel concept using its
existing subword repertoire.

\subsection{Perceptual Organ Models}

Beyond the frozen LLM (Broca's area), the architecture deploys a constellation
of frozen pre-trained specialist models as sensory organs:

\begin{itemize}[nosep]
  \item \textbf{Visual cortex} (DINOv2-Large, 307M): general-purpose visual
    features via self-supervised ViT.
  \item \textbf{Ventral stream} (I-JEPA ViT-H, 632M): semantic visual features
    via Joint Embedding Predictive Architecture---predicts in representation
    space, not pixel space.
  \item \textbf{Dorsal stream} (V-JEPA2 ViT-H, 632M): temporal dynamics and
    world prediction from video---LeCun's world model instantiated.
  \item \textbf{Spatial cortex} (Depth Anything V2, 335M): monocular depth
    estimation for spatial reasoning.
  \item \textbf{Auditory cortex} (Whisper-large-v3-turbo, 809M): speech
    transcription and audio embeddings.
  \item \textbf{Association cortex} (ImageBind, 1.13B): six-modality shared
    embedding space for cross-modal binding.
  \item \textbf{Extraction} (GLiNER2, 205M): zero-shot NER, relation extraction,
    and classification---replaces LLM-based few-shot prompting at 50$\times$
    lower latency.
  \item \textbf{Affect} (GoEmotions RoBERTa, 125M): 28-label multi-label emotion
    detection providing cognitive state signals (curiosity, confusion,
    annoyance) to the active-inference loop.
\end{itemize}

All organs are frozen, loaded lazily, and communicate with the substrate via
uniform \texttt{OrganOutput} dataclasses carrying features, metadata, and
confidence.  The total memory footprint for all organs loaded simultaneously is
${\sim}10.7$\,GB at fp16---well within a single-GPU or M-series Mac budget.

\subsection{Algebraic Substrate Components}

Each algebraic component has a \emph{functional job title} within the
architecture:

\paragraph{VSA/HRR (zero-shot analogy).}
Holographic Reduced Representations bind concepts via circular convolution in
$O(d \log d)$.  A triple \texttt{(subject, predicate, object)} becomes a single
10,000-dim hypervector that can be mathematically unbound later without any
gradient update.  This is how the agent relates novel concepts on the fly.

\paragraph{Hopfield (memory retrieval).}
Modern Continuous Hopfield networks (Ramsauer et al., 2020) provide one-step
content-addressable retrieval.  They take noisy, degraded input vectors and
converge to the nearest stored pattern.  The VSA does binding; the Hopfield does
retrieval.

\paragraph{Hawkes (temporal intuition).}
A multivariate self-exciting point process provides the substrate's sense of
conversational ``heat.''  When a conversational event fires, the baseline
intensity spikes and decays exponentially---this is how the system maintains
fluid short-term conversational spacing rather than filling a context window
with static timestamps.

\paragraph{Conformal prediction (calibrated uncertainty).}
Split-conformal predictors wrap the LLM's outputs with a finite-sample coverage
guarantee.  When $|C| > 1$, the substrate raises a Fristonian ambiguity signal
and steers the LLM toward a clarifying question.

\paragraph{SCM (causal reasoning).}
A finite structural causal model supports exact \texttt{do($\cdot$)}
interventions, counterfactual queries, and automatic backdoor/frontdoor
identification.  It grows during REM sleep via the PC algorithm applied to
journal observations.

\paragraph{Active inference (decision-making).}
Two categorical POMDPs minimize Expected Free Energy.  The winning posterior
entropy modulates the LLM's sampling temperature: confused substrate $\to$
exploratory decoding; decisive substrate $\to$ near-greedy decoding.

\subsection{Swarm Communication}

Multiple \textsc{Mosaic} instances on a LAN communicate via UDP multicast
(\texttt{239.255.77.1:50077}, TTL=1).  Every local EventBus event flows to the
network; every received network event flows to the local bus.  Peers discover
each other via heartbeat (2\,s interval, 8\,s timeout).  There is no
orchestrator---nodes are peers that communicate freely, and the substrate
decides what to do with what it hears.

\subsection{Evaluation Framework}

We evaluate on three tiers:
\begin{enumerate}[nosep]
  \item \textbf{Standard NLP benchmarks}: length-normalized MC scoring and
    greedy generation, with paired vanilla/shell/mind comparison.
  \item \textbf{Architecture probes}: scripted prompts scoring speech-exact and
    answer-present metrics (bare LLM vs.\ full substrate).
  \item \textbf{Substrate benchmarks}: eight CPU-only tests verifying algebraic
    guarantees (belief revision, hypothesis masking, Simpson's paradox, memory
    fidelity, conformal coverage, VSA round-trip, Hopfield retrieval, active
    inference decision quality).
\end{enumerate}
