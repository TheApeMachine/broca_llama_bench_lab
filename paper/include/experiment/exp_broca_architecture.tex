\subsection{Broca Architecture Probes}
\label{subsec:exp-broca-arch}

\paragraph{Protocol.}
We compare the bare frozen language host (\texttt{meta-llama/Llama-3.2-1B-Instruct}) against the full Broca cognitive substrate on scripted evaluation cases. Each case presents a prompt to both conditions and scores the output on two metrics: \emph{speech-exact} (verbatim match against the substrate's reference utterance) and \emph{answer-present} (the correct content word appears anywhere in the output). The baseline arm disables all grafts; the enhanced arm activates the full stack (semantic memory, active-inference agents, causal SCM, and all three graft types).

Answer-present accuracy (a relaxed metric accepting any output that contains the correct content word) shows a similar pattern: baseline 0.0\% vs.\ enhanced 0.0\% ($\Delta = +0.000$).
